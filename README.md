# SharpAppSync
<center>
<img src="logo.png" width="200">
</center>

C# + WinUI3 + Windows.Graphics.Captureを利用した画面のキャプチャサンプルです。  


## Usage
右クリックでメニューを出してキャプチャを開始します。


## Build
* Windows11
* Visual Studio 2026
* .Net8
* WindowsSDK 10.0.19041.0
* [Microsoft.Graphics.Win2D](https://www.nuget.org/packages/Microsoft.Graphics.Win2D/1.3.2?_src=template)
* [System.Reactive](https://www.nuget.org/packages/System.Reactive/6.1.0?_src=template)
* [ReactiveProperty](https://www.nuget.org/packages/ReactiveProperty/9.8.0?_src=template)
